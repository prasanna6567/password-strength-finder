Project Overview

About This Project

A modern cybersecurity-focused web application for password quality analysis and breach awareness.

Mission

Help users make stronger password decisions through clear feedback, privacy-preserving breach checks, and practical security education.

Privacy & Security Model

  • No password storage on client or server.
  • Client-side SHA-256 hashing before lookup.
  • K-anonymity style prefix matching endpoint.
  • Mock breach corpus support for offline demonstration.

Technology Stack

  • Backend: Python + Flask
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Architecture: Multi-page templates + static assets
  • UX: Responsive layout, accessibility support, smooth transitions

Important: Any breach tool should be used with broader defenses like MFA, password managers, and security monitoring.
